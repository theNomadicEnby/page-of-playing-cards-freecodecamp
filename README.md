## Objective

- [x] Build an app that is functionally similar to [Build a Page of Playing Cards | freeCodeCamp](https://www.freecodecamp.org/learn/responsive-web-design-v9/lab-page-of-playing-cards/build-a-page-of-playing-cards).

## User stories
- [x] You should build a webpage that displays at least three playing cards.
- [x] You should have a `main` element with an ID of `playing-cards`.
- [x] Within your `#playing-cards` element, you should have at least three `div` elements, each with a class of `card`.
- [x] Within each `.card` element, you should have three `div` elements, the first with a class of `left`, the second with a class of `middle`, and the third with a class of `right`.
- [x] Your `#playing-cards` element should use flexbox to horizontally center its children, allow them to wrap, and put a `20px` space between them.
- [x] Each of your `.card` elements should use flexbox to justify its children using `space-between`.
- [x] Each of your `.left` elements should use flexbox item properties to align itself at the start of its' parent container.
- [x] Each of your `.middle` elements should use flexbox item properties to align itself in the center of its' parent container.
- [x] Each of your `.right` elements should use flexbox item properties to align itself at the end of its parent container.
- [x] Each of your `.middle` elements should use flexbox to display its children in a column.
